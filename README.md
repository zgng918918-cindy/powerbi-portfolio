Sales Performance Dashboard (Power BI)

This dashboard was designed to help management monitor revenue performance, identify trends, and analyze product and regional contributions using interactive filters and time-based KPIs.

What this dashboard shows
	•	Overall sales and order volume
	•	Year-to-date (YTD) performance
	•	Sales by product category
	•	Transaction filters by month and region

Key technical features
	•	Built a star-schema data model (Orders fact table with Customers, Products, and Calendar dimensions)
	•	Created a custom Calendar table to support time intelligence
	•	Developed DAX measures including Total Sales, YTD, MoM, and rolling comparisons
	•	Designed an interactive dashboard with slicers and drill-down capability

Tools used
	•	Power BI Desktop
	•	Power Query (data transformation)
	•	DAX (time intelligence and KPI calculations)
